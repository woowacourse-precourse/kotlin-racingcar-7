# kotlin-racingcar-precourse

### 입력

-   자동차 이름 ("," 기준으로 구분, 5자 이하, 영대소문자 & 숫자만 가능)
-   시도할 횟수 (0 이상의 수)
-   잘못된 값 입력시 IllegalArgumentException 발생

### 출력

-   차수별 실행 결과 (자동차 이름 : 이동거리 - "\n"으로 구분)
-   우승자 안내 문구 (자동차 이름 - ", "으로 구분)

### 자동차

-   무작위 값 : 0 ~ 9 사이의 무작위 값
-   전진 : 무작위 값 4 이상
-   멈춤 : 무작위 값 4 미만
