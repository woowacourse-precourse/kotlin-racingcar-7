# kotlin-racingcar-precourse
# 자동차 경주

----------

## 기능 목록

### 1. 입력

- 이름 입력 및 쉼표 기준 분리
- 시도 횟수 입력
- 이름 5자 이하인지 확인

### 2. 경주 기능

- 무작위 값이 4 이상인지 판별
- 각 자동차의 전진 횟수를 저장
- 전진 횟수 비교하여 최종 우승자 도출

### 3. 출력
- 실행 결과 출력
- 최종 우승자 출력

### 4. 예외 처리
- 이름이 5자 초과인 경우
- 이름이 0자인 경우
- 참여 인원이 1명 이하인 경우
